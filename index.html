블록과 인라인

요소성질

블록성질 : h1~h6(headline tag - Title part), p(paragraph - 문단), div(divide - 공간을 형성, 그룹핑 작업)

- 줄바꿈이 존재

- 공간에 대한 구성이 정확하게 가로값과 세로값으로 떨어진다.

- 독립적인 말단 태그(h1~h6), 복합적인 태그(div)

블록성질 => As like a lego block(상하로만 쌓임 =>세로방향으로 나열)

인라인성질 : a (anchor - 링크를 걸어줄 떄 사용 / href라는 명칭의 속성을 반드시 넣어주어야 합니다.), img(image 넣을 때 / sec(source), / alt(alternative- 시각장애인을 위한 이미지의 설명)), span (인라인 성질을 갖고 있는 독립적이면서도 그룹핑이 가능)

- 줄바꿈이 없음

- 자체 커서영역이 존재(통제대상) : HTML 문서상에서 엔터값이 화면상에 반영

- 공간에 대한 세로영역이 모호

- 독립적인 말단 태그(a, img, span), 복합적인 태그(span)

인라인 성질 => 인라인 스케이트의 바퀴를 연상하면 됨.
